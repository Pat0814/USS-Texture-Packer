# USS Texture Packer v1.0.0

USS Texture Packer is a free standalone companion utility for packing linear grayscale maps into practical RGBA texture outputs.

It is designed for clean linear grayscale-to-RGBA packing workflows and ships with ready-to-use presets for:

- `ORM`
- `ORD`
- `MRD`
- `RMA`
- `MRA`
- `ARM`
- `Masks`
- `Custom`

## Included

- `USS_TEXTURE_PACKER.exe`
- UI screenshot
- Quick start guide
- release metadata
- version file

## Notes

- This is a standalone Windows utility.
- It is optional and not required to use Ultra Surface Shader.
- The tool is provided as a free external companion release.
