# USS Texture Packer v1.0.0

USS Texture Packer is a free standalone companion utility for packing linear grayscale maps into practical RGBA texture outputs.

It is designed for clean linear grayscale-to-RGBA packing workflows and ships with ready-to-use presets for:

- `ORM`
- `ORD`
- `MRD`
- `RMA`
- `MRA`
- `ARM`
- `Masks`
- `Custom`

## Related USS Product

This utility was created as a free companion tool for the USS workflow and pairs well with [USS - Ultra Surface Shader](https://www.fab.com/listings/254b7083-5a3a-4f7f-bcf5-2029b442169d) on Fab.

Ultra Surface Shader is the main USS product: a modular Unreal Engine shader system built around three master materials for base surfaces, cloth response, and layered blend workflows.
