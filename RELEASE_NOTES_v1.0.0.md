# USS Texture Packer v1.0.0

Initial public companion release.

## Included

- standalone Windows executable
- built-in presets for `ORM`, `ORD`, `MRD`, `RMA`, `MRA`, `ARM`, `Masks`, and `Custom`
- USS-aligned branding and icon
- production-ready UI screenshot

## Notes

- This release is intended as a free external companion utility.
- It is not required to use Ultra Surface Shader.
- Binary downloads are provided through GitHub Releases, not through the repository contents.
